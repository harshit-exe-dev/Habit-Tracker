Habit Tracker

A simple Habit Tracker web application built using HTML, CSS, and JavaScript.

Live Demo

🔗 https://harshit-exe-dev.github.io/Habit-Tracker/

Features

- Add new habits
- Mark habits as completed
- Track streaks
- Progress bar visualization
- Delete habits
- Dark mode toggle
- Data saved using LocalStorage

Technologies Used

- HTML
- CSS
- JavaScript
- LocalStorage
